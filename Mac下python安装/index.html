M1芯片安装步骤

相较于 Intel芯片的安装，M1/M2系列的安装略显麻烦点儿。
首先是了解下两个芯片的环境区别：

  • Mac默认终端是bash，操作的是～.bashrc或者~.bash_profile，老款都是Intel芯片。
  • M1的默认终端是zsh，所以操作的是～.zshrc或者～.zprofile；在这里面配置的，作为命令时才能被识别；M1是ARM架构芯片

设置Intel环境方式运行终端

访达->其他工具–终端–右键显示简介–Rosetta
用uname -p确认当前的环境是i386,然后就可以继续下面步骤了。
